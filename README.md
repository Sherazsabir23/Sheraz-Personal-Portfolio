## 🚀 Installation & Setup
1. Clone the repository:
   ```sh
   git clone https://github.com/Sherazsabir23/Sheraz-Personal-Portfolio.git
   ```
2. Navigate to the project folder:
   ```sh
   cd Sheraz-Personal-Portfolio
   ```
3. Open `index.html` in a browser.
